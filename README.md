## Setting Up the Extension: Getting It to Work the Way You Want

After installing, spend two minutes in the settings before you start using it. Here's what's worth adjusting:

**Icon display timing** — Go into Basic Settings and choose how the PikPak icon appears next to links:
- *Always show*: The icon sits there permanently next to every resource link. Good if you're actively hunting for files.
- *Hover*: Only shows when you mouse over a link. Less visual noise.
- *Select*: Only appears when you actually highlight the link text. Most unobtrusive option.

**Clipboard monitoring** — If you're frequently copying download links from places the extension can't directly see (Discord messages, other apps), turning this on means you just copy the link and the extension catches it automatically.

**Account connection** — Make sure you're signed in. The extension needs your account to know where to send the files.

## What the Extension Can't Do (Honest Caveats)

It's worth being straightforward about the limits:

- **Not all links get detected**: The extension is good at identifying common resource link formats (magnets, torrents, direct download URLs). It won't catch everything on every site — some platforms use obfuscated or JavaScript-rendered links that the extension can't see without you manually copying and submitting them.
- **Free tier limits bite quickly**: 6GB disappears fast if you're dealing with video files. The extension works fine on free, but you'll hit the storage ceiling quickly.
- **One CSS quirk**: Some users have noted the extension injects CSS variables into pages, which can occasionally interfere with site styling. It's minor and fixable by adjusting which sites the extension runs on, but worth knowing.
- **No offline functionality**: The extension is a pipe to PikPak's servers. If you can't reach PikPak, the extension can't do anything.
